# Rally Technology Website

企業官網重建專案，技術基線為 Nuxt 4 + Nuxt UI + Vue 3 + TypeScript。

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build / Preview

```bash
npm run build
npm run preview
```

## Quality Checks

```bash
npm run format:check
npm run lint
npm run typecheck
```
