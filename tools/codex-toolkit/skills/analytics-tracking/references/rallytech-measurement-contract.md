# RallyTech Measurement Contract

Use this template when the implementation is already stable and you need a concise tracking plan.

```text
Page / Flow:
Business goal:
Primary conversion:
Secondary conversion:

Object | Event name | Trigger | Parameters | Destination | Conversion? | Validation note
```

## Recommended Objects

- hero CTA
- sticky CTA if present
- contact form start
- contact form success
- service-card or inquiry-card click
- trust or proof interaction if it affects conversion decisions
- outbound contact actions

## Notes

- Keep event names stable even if visible labels change.
- Prefer a short contract that the team will maintain.
- Add implementation owner if multiple surfaces share the same event.
