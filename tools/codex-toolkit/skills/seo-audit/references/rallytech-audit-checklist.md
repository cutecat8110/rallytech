# RallyTech Audit Checklist

Use this checklist when the task is a launch-readiness or post-implementation SEO audit.

## Order

1. crawlability
2. indexable URL shape
3. metadata and canonicals
4. heading and body alignment
5. internal links and breadcrumbs
6. performance and RWD
7. schema handoff

## Output Shape

- blocker
- warning
- evidence
- affected page or template
- recommended owner

Focus on the few issues most likely to block discovery, indexing, or CTR.
