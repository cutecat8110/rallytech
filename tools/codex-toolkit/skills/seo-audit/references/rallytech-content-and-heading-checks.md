# RallyTech Content And Heading Checks

Use this when metadata is fine but page intent or heading structure still feels weak.

## Review Questions

- Is there one clear H1?
- Do H2s map to real block purposes?
- Does the visible copy support the page's target intent?
- Are service claims and proof signals visible in the body?
- Does bilingual copy preserve the same message hierarchy?

## Common Problems

- multiple H1s
- headings used only for styling
- meta promises not supported by body content
- generic translation that weakens intent

If this becomes a rewrite problem instead of an audit problem, hand off to the relevant copy or page-planning skill.
