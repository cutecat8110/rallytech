# RallyTech Performance And RWD Checks

Use this reference when layout quality and load behavior affect SEO signoff.

## Check

- oversized images or videos
- blocking CSS or script bundles
- layout shift in hero or nav
- mobile viewport overflow
- unreadable text density on small screens
- tap targets that are too small or too close

## Reporting

Keep findings practical:

- what the user sees
- which template or component likely owns it
- whether it is a blocker, warning, or polish item
