# Context Sections

## 建議優先順序

1. Product overview
2. Target audience
3. Problems and pain points
4. Differentiation
5. Objections
6. Customer language
7. Proof points
8. Goals

## 每個 section 要抓什麼

### Product Overview

- 一句話描述
- 產品 / 服務類型
- 商業模式

### Target Audience

- 目標公司或客群
- 主要決策者 / 使用者
- 主要 use case

### Problems and Pain Points

- 客戶在找你之前最痛的是什麼
- 現有替代方案為什麼不夠好
- 代價是時間、金錢、機會，還是壓力

### Differentiation

- 你比替代方案多做了什麼
- 為什麼這個差異對客戶真的重要

### Objections and Anti-Persona

- 最常見的 3 個 objections
- 哪些人其實不適合

### Customer Language

- 客戶怎麼描述自己的問題
- 客戶怎麼描述你帶來的結果
- 應使用與避免的詞

### Proof Points

- 客戶、案例、數字、testimonial
- 哪些證據可被公開使用

### Goals

- 主要商業目標
- 希望使用者採取的 conversion action
