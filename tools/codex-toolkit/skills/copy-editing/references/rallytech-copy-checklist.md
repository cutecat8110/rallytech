# Copy Checklist

## 快速檢查

- 核心訊息有沒有被改掉
- CTA 是否更清楚，而不是更模糊
- 語氣是否仍像同一家公司
- proof 是否被保留或標註待確認
- 是否出現新的誇大說法

## 常見問題

### 只有 feature，沒有 benefit

- 補上「所以對讀者有什麼改變」
- 把 capability 轉成 outcome

### 太像企業空話

- 把抽象用詞換成真實說法
- 若一句話不能被具體解釋，通常需要重寫

### 開頭太弱

- 不要從公司自我介紹開始
- 先講讀者問題、任務或關鍵結果

### 沒有 proof

- 補 testimonial、數字、案例、流程證據
- 若沒有證據，降低斷言強度

### 太多 audience 一起講

- 決定主要讀者
- 讓句子只服務主要讀者的決策

### 只修字，沒有修理解

- 重新檢查 Clarity sweep
- 優先解決不懂，再追求漂亮
