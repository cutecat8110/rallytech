# RallyTech Finding Template

## Findings

- Severity: high / medium / low
- Location: page, block, component, or state
- Issue: what is wrong
- Impact: why this affects clarity, trust, conversion, or fidelity
- Recommendation: the smallest clear fix

## Open Questions or Assumptions

- Note only items that materially affect the judgment.

## Change Summary

- Keep this short.
- Use only after listing findings.
