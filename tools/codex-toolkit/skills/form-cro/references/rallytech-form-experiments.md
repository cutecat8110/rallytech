# Form Experiments

只列與目前 bottleneck 直接相關的假設，不要為了測試而測試。

## 常見實驗方向

### Form Structure

- 單步驟 vs 多步驟
- 單欄 vs 雙欄
- 內嵌頁面 vs 獨立頁

### Field Scope

- 最小必填欄位組合
- `phone` 必填 vs 選填
- `company` 顯示 vs 延後詢問

### CTA / Trust

- CTA wording
- response time 文案
- privacy assurance 文案
- form 附近的 proof placement

### Mobile UX

- 更大的 tap target
- sticky CTA
- 對應 keyboard type
- 降低首屏欄位密度
