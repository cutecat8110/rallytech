# RallyTech Schema Priority

Use this order unless the page clearly requires something else:

1. `Organization`
2. `WebSite`
3. service-like page schema when visible content clearly supports the service meaning
4. `FAQPage`
5. `BreadcrumbList`

## Why This Order

- organization and website data establish core site identity
- service-like pages are only useful when the page body is specific and trustworthy
- FAQ schema should reflect real visible FAQ content
- breadcrumbs should follow the real information hierarchy

Avoid adding schema purely because a rich result sounds attractive.
