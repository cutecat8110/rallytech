# RallyTech Validation And Rich Results

Validate every payload before calling the work done.

## Tools

- Google Rich Results Test
- Schema.org Validator
- rendered DOM inspection in browser

## Must Check

- required properties exist
- URLs are absolute and canonical
- FAQ answers match visible content
- breadcrumb order matches the page hierarchy
- organization data matches site-wide truth

## Reporting

Return:

- schema type checked
- validation method
- pass or fail
- blocker or warning
- next fix owner if needed
