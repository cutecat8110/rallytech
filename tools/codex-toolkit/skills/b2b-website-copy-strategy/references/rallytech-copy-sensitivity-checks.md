# RallyTech Copy Sensitivity Checks

## Core Checks

- clarity: can a busy engineering buyer understand this quickly
- benefit: does it explain why the reader should care
- proof: is the claim supported
- specificity: is it concrete enough
- CTA fit: does the action match the page job

## Red Flags

- generic agency language
- invented metrics or credentials
- direct translation tone in `zh-tw`
- repeating the same promise without deeper proof
