# RallyTech Promotion Cases

## Promote to `main.css`

- shared token
- semantic CSS variable
- shared layout shell
- shared primitive used across multiple sections

## Promote to `app/app.config.ts`

- cross-page Nuxt UI theme bridge
- stable component slots / variants
- semantic color mapping

## Keep in `<UTheme>` or `:ui`

- one page or one block needs a different Nuxt UI tone
- the change is structural but not global

## Keep local

- single-section spacing
- crop, overlay density, breakout geometry
- one-off layout grammar
