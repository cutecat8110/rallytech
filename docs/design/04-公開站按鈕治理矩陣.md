# 公開站按鈕治理矩陣

最後更新：2026-04-24

## 文件目的

- 定義公開站按鈕的唯一 taxonomy，避免同一語意出現多套 hover、陰影、尺寸與 state 規則。
- 補齊 `app/app.config.ts`、`app/utils/button-themes.ts` 與 `/design-system` 之間的治理對照。

## 適用範圍

- 公開站 frontend pages：`/`、`/about`、`/services`、`/services/*`、`/one-touch-experience`、`/contact`
- `/design-system` 與 `/design-system-color-lab` 僅作 internal preview 與 evidence，不等於公開站 contract。

## 核心原則

- `primary solid` 是唯一品牌主 CTA。
- 公開站只保留 5 種可感知 archetype：`primary solid`、`contrast solid`、`segmented control`、`icon utility`、`text CTA`。
- `app/app.config.ts` 只承接全站 size ladder 與 `primary solid`。
- 其他公開站按鈕變體必須落在 `app/utils/button-themes.ts` 的 approved `UTheme` family。
- `SharedActionLink` 是 card/list 的 text CTA primitive，不升級為 `UButton`。
- local class 只負責寬高、排列、`min-width`、區塊 spacing；不得重複定義跨區塊的背景、文字色、hover、focus、active、shadow 或 `!important`。

## Public Taxonomy

| Archetype           | Owner layer                           | 使用場景                                                                      | 尺寸規則                           | 動效規則                                                                            |
| ------------------- | ------------------------------------- | ----------------------------------------------------------------------------- | ---------------------------------- | ----------------------------------------------------------------------------------- |
| `primary solid`     | Layer 2 `app/app.config.ts`           | 首頁 hero、首頁 section CTA、contact submit                                   | 預設 `lg`，首頁 hero 用 `xl`       | hover 只做背景色階切換；active 用較深色；focus 用品牌 ring                          |
| `contrast solid`    | Layer 3 `button-themes.ts` + `UTheme` | Header contact、footer connector                                              | Header `sm`，footer connector `lg` | 深底版本維持深色實底；白底版本 hover 會染上淡主色，文字與 focus ring 同步切到主色系 |
| `segmented control` | Layer 3 `button-themes.ts` + `UTheme` | Header FREE/AI、語系切換                                                      | 預設 `xs`                          | 未選取白底項 hover / focus 會切到淡主色底與深主色文字；已選取實底維持現況           |
| `icon utility`      | Layer 3 `button-themes.ts` + `UTheme` | Header mobile controls、contact quick actions、newsletter submit、back-to-top | 預設 `sm`                          | inline 與 emphasis 模式都只做 fill、text 的細微切換；focus 才加強 ring              |
| `text CTA`          | Shared primitive                      | Home/services cards 的文字箭頭提示                                            | 預設 `sm`                          | 只做文字與 icon 色彩加深；不使用位移、box、shadow、fill                             |

## Size Ladder

| Size | 用途                                                         |
| ---- | ------------------------------------------------------------ |
| `xs` | 僅限 segmented control 與 header utility control             |
| `sm` | Header contact、icon utility、密集工具列                     |
| `md` | 一般內文型 standalone CTA 或單欄 mobile action；公開站不主打 |
| `lg` | 預設 section CTA、footer connector、contact submit           |
| `xl` | 僅首頁 hero 主 CTA                                           |

## Section Mapping

| 區塊 / 元件                               | 目標 archetype      | Size | Owner                            | 備註                                                     |
| ----------------------------------------- | ------------------- | ---- | -------------------------------- | -------------------------------------------------------- |
| Header contact                            | `contrast solid`    | `sm` | `contrastSolidDarkButtonTheme`   | 保留公開 CTA，但不再借用 icon/menu 語氣                  |
| Header FREE/AI、語系                      | `segmented control` | `xs` | `segmentedControlButtonTheme`    | 維持公開，但不得升格成 CTA                               |
| Header mobile menu、mobile service toggle | `icon utility`      | `sm` | `iconUtilityLightButtonTheme`    | 白底 inline utility，不使用 solid fill                   |
| Home hero CTA                             | `primary solid`     | `xl` | Global `UButton`                 | 唯一 hero 級主 CTA                                       |
| Mission / One-touch / Contact submit      | `primary solid`     | `lg` | Global `UButton`                 | section-level 主 CTA                                     |
| Footer connector                          | `contrast solid`    | `lg` | `contrastSolidLightButtonTheme`  | 深底白按鈕                                               |
| Newsletter submit                         | `icon utility`      | `sm` | `iconUtilityDarkButtonTheme`     | dark surface inline icon action                          |
| Contact quick actions                     | `icon utility`      | `sm` | `iconUtilityLightButtonTheme`    | light surface inline icon action                         |
| Back to top                               | `icon utility`      | `sm` | `iconUtilityEmphasisButtonTheme` | 使用較強 fill 建立存在感，但仍維持 flat chromatic motion |
| Home / services cards                     | `text CTA`          | `sm` | `SharedActionLink`               | card/list link hint，不改成 `UButton`                    |

## Internal-only Themes

以下變體只保留在 internal preview 與色彩實驗，不得進公開站 production contract：

- `internalDarkGhostButtonTheme`
- `internalOutlineButtonTheme`
- `internalAccentSolidButtonTheme`

這些變體可出現在 `/design-system` 或 `/design-system-color-lab`，但不得回流到 frontend 公開頁。

## 驗證要求

實作或調整按鈕治理後，至少完成以下驗證：

1. `npm run format:check`
2. `npm run lint`
3. `npm run guard:ui-buttons`
4. `npm run typecheck`
5. Desktop / mobile 檢查 header、首頁 hero CTA、one-touch CTA、contact submit、footer connector、newsletter submit、back-to-top
6. 互動抽查 public buttons hover 前後不得出現 translate 與 box-shadow 升級；`SharedActionLink` 不得有箭頭位移

## 對應 runtime truth

- Global standard：`app/app.config.ts`
- Approved public themes：`app/utils/button-themes.ts`
- Visual evidence：`app/pages/design-system.vue` 的 `ButtonsSection`
- Internal experiments：`app/pages/design-system-color-lab.vue`
