## 依需求查找

- 想開始切版：
  先看 `design/01-設計規則整理.md`，再看 `design/02-AI切版規則.md`
- 想知道網站架構與頁面定位：
  看 `project/02-網站資訊架構.md`
- 想知道某個服務命名或分類為什麼這樣定：
  看 `project/04-服務架構決策.md`
- 想回頭查舊官網內容：
  看 `project/05-舊站內容盤點.md`
- 想看參考站整理：
  看 `references/README.md`
- 想確認怎麼寫 commit 與 PR title：
  看 `project/06-Commit-規則與格式.md`
