# 網站資訊架構設計簡版

最後更新：2026-03-16

## 專案定位

- 雷力科技企業官網重構
- 主要受眾為具技術背景的專業客戶
- 網站語氣以專業、清楚、可信賴為主
- 首波提供中英文雙語

## 設計目標

- 讓使用者快速理解雷力提供哪些服務
- 清楚區分 `服務項目` 與 `一站式整合`
- 保留工業自動化專業感，不過度白話
- 導向 `聯絡我們` 作為主要轉換行為

## 第一層 IA

### 概念架構

- 首頁
- 關於我們
- 服務項目
- 一站式整合
- 聯絡我們

### 桌機主選單

- 關於我們
- 服務項目
- 一站式整合
- 聯絡我們

### 導覽規則

- `首頁` 不放在桌機主選單
- Logo 可點擊回首頁

## 首頁區塊清單

- Hero
  - 主標
  - 簡短輔助文案
  - 主要行動按鈕
- 公司定位
  - 公司角色說明
  - 自動化 / 儀控整合定位
- 核心優勢
  - 3 個重點欄位
  - 建議方向：專業經驗 / 技術應用 / 合作整合
- 服務總覽
  - 區塊標題
  - 8 項服務入口
- 專案流程
  - 4 步驟流程
  - 需求確認 / 系統規劃 / 專案導入 / 驗證與優化
- 一站式整合
  - 區塊標題
  - 5 項整合內容入口
  - 次要行動按鈕
- 品牌行動按鈕區塊
  - 品牌主張或 mission
  - 導向服務或進一步了解
- 聯絡行動按鈕區塊
  - 聯絡引導
  - 導向聯絡我們

## 服務項目

- SCADA and HMI Graphics
- I&E Services
- PLC & DCS Programming and Migration
- PI Server
- Historians
- Network Design
- Remote Monitoring and Data
- Alarm Monitoring

## 一站式整合

- Data Integration
- Call-Out System Integration and Commissioning
- Vibration Analysis
- Local/Cloud Historians Deployment
- Manure Management System Integration

## 命名方向

- 中文介面可搭配英文專有名詞
- 命名以專業、短、好掃讀為原則
- `One Touch Experience` 中文暫定為 `一站式整合`

## 設計備註

- 本版先以參考網站內容架構為主
- 舊官網內容暫不作為首輪設計主依據
- `首頁` 區塊清單已依參考網站首頁整理
- `關於我們`、`服務項目`、`一站式整合`、`聯絡我們` 的內頁區塊仍待逐頁討論
- 若需完整背景、命名討論或 IA 細節，請再參考其他規劃文件
