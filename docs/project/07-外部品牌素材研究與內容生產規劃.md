# 外部品牌素材研究與內容生產規劃

最後更新：2026-04-15

## 文件目的

- 整理雷力科技目前可從網路上驗證的外部品牌素材。
- 區分哪些資訊可直接支撐官網文案，哪些只能作為線索或需要內部再確認。
- 把「可直接上站的 proof」與「需要補訪談 / 製作的新內容」拆開，方便後續內容生產。

## 研究範圍

- 現行官方網站與其站內服務頁
- 舊官網與 Wayback Archive 可辨識資訊
- 官方對接頁面與合作夥伴頁
- 公司登記與公開變更資訊
- 招募與公司資料平台上的次級資訊

## 可驗證外部素材

### 1. 公司定位與基本敘述

可直接採用的核心事實：

- 雷力科技成立於 `2014`。
- 官方英文首頁將公司定位為 `Your Intelligent System Consultant`。
- 官方中英首頁皆寫明公司專注於 `automation control and monitoring systems` 的技術研發與整合服務。
- 官方首頁明確主張雷力是 `台灣首家本土投入離岸風力發電系統整合的企業 / the first local company in Taiwan dedicated to offshore wind power system integration`。

目前最穩的品牌主軸：

- 工業自動化控制與監控系統整合
- 再生能源與離岸風電
- 專案工程與系統架構規劃
- 客製化系統開發

### 2. 品牌價值與企業精神

官方首頁可直接支撐的價值語彙：

- `專業 / Professionalism`
- `迅速 / Promptness or Efficiency`
- `團隊 / Teamwork`
- `誠信 / Trustworthiness or Integrity`

這組價值觀已同時出現在官方中英首頁與 104 公司頁，可視為目前最穩定的品牌價值基礎。

### 3. 信任與資格 proof

官方與官方對接來源可支撐的 proof：

- 官方首頁明列通過 `ISO 9001` 與 `ISO 45001`。
- sitemap 可見 2025 年上傳的 `SGS ISO 9001` 與 `SGS ISO 45001` 圖像資產。
- WinCC OA Partner 頁列出雷力科技為合作夥伴，並揭露：
  - 業務重點：以本國業務為主，亦參與國際專案
  - 參考案名：
    - `CIP (Copenhagen Infrastructure Partners) Offshore Windfarm Project`
    - `CPC EMS Power Storage Project`
    - `Kaohsiung Port Energy Management Project`
  - 分支領域：
    - `Energy`
    - `Industry`
    - `Oil & Gas`
    - `Traffic & Transport`

### 6. 公司登記與歷史節點

目前能從網路公開資料交叉驗證的歷史節點：

- 多個公司資料鏡像與搜尋結果皆顯示設立日期為 `2014-08-26`。
- 官方首頁使用 `Founded in 2014`，與外部資料一致。
- 官方 PDF 搜尋結果可辨識以下節點：
  - `2020-12-14`：公司所在地為 `新竹縣竹北市光明六路東一段253號5樓`
  - `2022-09-21`：公司所在地變更為 `新竹縣竹北市惟馨街95號3樓之5`
  - `2024-08-16`：同址資本額變更，搜尋摘要顯示 `36,742,940`

注意：

- `資本額` 在 104、官方 PDF 搜尋摘要與多個鏡像站之間不完全一致。
- 若未來想把資本額、代表人、董監事或詳細時間線放上官網，建議改由你們內部提供最新公司登記謄本，不要直接採信單一外站。

### 7. 更早期官網狀態

Wayback 可確認：

- `2017-2019` 年間存在較早期的 ASP.NET 官網。
- 至少可辨識下列路徑：
  - `index.aspx`
  - `index.aspx?id=1`
  - `index.aspx?id=2`
  - `index.aspx?id=3`
  - `index.aspx?id=4`
  - `Work.aspx`
  - `blog.aspx`

但目前 archive 的文字可讀性很差，多數只能抓到頁標 `雷力科技有限公司`，無法穩定萃取可直接引用的內容。因此：

- 可以確認「更早期站點存在」這件事
- 但不能把舊 ASP.NET 站當作高品質文字來源

## 可直接用於官網的內容生產

### A. 低風險，可直接生產

這些內容已有 `A1 / A2 / A3` 級來源支撐，可直接納入下一輪網站內容規劃：

| 內容模組     | 可用素材                                            | 適合頁面                      |
| ------------ | --------------------------------------------------- | ----------------------------- |
| 公司簡介短版 | 2014 成立、自動化控制與監控系統整合、智能化系統顧問 | 首頁、頁尾、聯絡我們          |
| 品牌價值     | 專業、迅速、團隊、誠信                              | 首頁、關於我們                |
| 信任徽章     | ISO 9001、ISO 45001、WinCC OA Partner               | 首頁、關於我們                |
| 產業覆蓋     | 再生能源、風場、化工、製藥、EPC、工業               | 首頁、服務總覽                |
| 技術平台     | SCADA、WinCC OA、AVEVA、Historian                   | 服務總覽、服務內頁            |
| 專案領域     | 離岸風電、儲能、能源管理                            | 首頁、再生能源相關 proof 區塊 |

### B. 中風險，需要內部確認再公開

這些內容很有價值，但目前仍建議先經內部確認：

- `台灣首家本土投入離岸風力發電系統整合的企業`
  - 來源是官方首頁，可用於對外，但若要放大成 hero claim 或 SEO title，建議先確認法務與業務是否願意承擔競品比較風險。
- `資訊安全 / cybersecurity`
  - 出現在 104 公司頁，但目前官方網站幾乎沒有展開。
- `智慧製造 / 公部門`
  - 出現在 104 公司頁，若要上站建議至少要有一個對應案例、場景或服務模組可承接。
- `CIP / CPC / 高雄港` 三個案名
  - 目前可從 WinCC OA Partner 頁看到，但若要直接上站，建議先確認客戶名稱是否可公開使用。

### C. 高價值，但需要新製作

這些是目前最值得補，但外部網路資料無法完整生產的內容：

- 公司里程碑時間線
- 創立故事或創辦理念
- 團隊介紹與跨域角色
- 代表案例頁
- 客戶證言
- 認證與安全管理專頁
- 客戶 / 產業 logo wall

## 建議內容生產路線

### Phase 1：先把外部 proof 收進站內

目標：

- 不再只靠抽象文案傳遞專業
- 先用已驗證素材補足信任感

建議產出：

- 首頁新增 `Trust / Proof` 區塊
  - `Founded in 2014`
  - `ISO 9001 / ISO 45001`
  - `Offshore Wind Integration`
  - `SCADA / Automation / Historian`
- 關於我們新增 `Brand Foundations` 區塊
  - 專業、迅速、團隊、誠信
- 服務總覽新增 `Industries We Support` 或 `Common Project Contexts`
  - 再生能源
  - 風場
  - 化工
  - 製藥
  - EPC

### Phase 2：補訪談，補能說服人的品牌內容

建議內部訪談題目：

1. 離岸風電系統整合的起點是什麼？
2. `專業、迅速、團隊、誠信` 在實際專案中分別代表什麼？
3. 客戶為什麼選雷力，而不是一般 SI 或設備商？
4. 哪 3 類專案最能代表雷力？
5. 哪些客戶名稱、產業類型、專案模組可以公開？

### Phase 3：生產可長期使用的品牌資產

建議優先製作：

- `公司里程碑年表`
- `3 篇代表案例`
- `1 頁認證與安全管理說明`
- `1 版品牌 boilerplate`
  - 50 字版
  - 100 字版
  - 250 字版

## 目前仍不建議直接上站的說法

- 未經確認的資本額
- 未取得公開授權的客戶名稱與案例細節
- `資訊安全` 若無對應服務頁與實績，不建議直接升為主定位
- 招募頁上的企業文化長文，不建議直接搬進官網
