# 首頁媒體素材來源

最後更新：2026-04-23

## Owner boundary

這份文件現在是 appendix only。
它保留人類可讀的素材來源、授權與選圖脈絡，但不再作為 `public/images` 的 canonical inventory owner。
真正的 canonical inventory、命名與 lifecycle status 請以 `data/image-assets/catalog.json` 為準。

## 選圖原則

- 不再以泛工業現場照作為首頁主視覺。
- 改以參考站的四種圖片角色反推變體：
  1. `Hero`：高張力控制室 / 監控空間
  2. `About`：高科技建築或現代設施立面
  3. `OTE`：機櫃、伺服器或資料整合場景
  4. `Mission`：單一技術角色 / 現代工程工作情境
- 外部 stock 素材皆為可商用開放授權來源，並本地化存放於 `public/images/stock/home/`。
- 公司實拍素材為 RallyTech 內部自有照片，維持於 `public/images/company/office/`，不歸入外部 stock canonical path。
