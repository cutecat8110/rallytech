# 關於我們合作夥伴素材來源

最後更新：2026-04-24

## Owner boundary

本文件記錄 About 頁合作夥伴區塊的視覺資產來源與使用限制。Canonical inventory 仍以 `data/image-assets/catalog.json` 為準。

## 資產策略

- 本輪不直接下載第三方官方 logo 檔案。
- 本輪建立四張 site-local SVG badge，統一比例、留白、線條與色彩，避免不同來源 logo 造成風格不一致。
- SVG badge 僅用於官網平台辨識與合作生態表述；各品牌名稱與商標仍歸原權利人所有。
- 對外文字以 `WinCC OA Partner` 作為可驗證 proof，其他品牌與產品維持「平台與技術生態」語氣。
