# references 對照導覽（跨部門共用）

這份導覽提供管理部、業務、老闆、設計師與工程團隊共用的對照方式，目標是快速看懂「參考站怎麼對照」，並直接回到檔案修改 `en` 與 `zh-tw`。

## 三層檔案（固定結構）

- `source/*`：來源原文（含來源異常註記）
- `en/*`：英文可讀稿（可直接調整）
- `zh-tw/*`：繁中可用稿（可直接調整）

## 3 步驟快速操作

1. 先看 `IA-參考站與目標站對照.md`，找到要處理的頁面編號。
2. 用同一編號開啟三份檔案：`source`、`en`、`zh-tw`。
3. 以 `source` 為對照基準，直接修改 `en` 與 `zh-tw`。

## 修改邊界（精簡版）

- 保留來源語意，不逐字照抄來源句型。
- 排除模板殘留文字（例如 Construction、Blog、lorem ipsum）。
- 不得保留他站公司資訊（電話、信箱、地點、公司名）。
- 遇到矛盾或異常時，先看該 `source` 檔案底部的「來源異常註記」再決定改法。

## 對照規則（編號優先）

- 同一頁一律用同一編號對照：`source/xxx.md` ↔ `en/xxx.md` ↔ `zh-tw/xxx.md`
- 檔名可因語言不同，但編號必須一致（00、01、02...）
- 內容判斷不確定時，以 `source/*` 為最終依據
